# Running the Project Locally

Clone the repository:

```bash
git clone https://github.com/Silas-N-Schlax/rm-prework-odin-mastermind.git
```

Navigate to the project directory:

```bash
cd rm-prework-odin-mastermind
```

```bash
bundle install
```

Run `ruby main.rb` in your terminal.
